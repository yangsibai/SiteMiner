Features

You will definitely like Mockup Plus and its nice features.

  • Simple and quick

    There is no need for any learning or learning cost. You will be able to quickly master Mockup Plus. What you need to do is just drag the mouse and set some attributes. Simple and quick operation help you focus on design, rather than concentrate on the learning of using a design tool...

  • Adequate component library

    With the built-in 100 multiple components, you can design any software prototype. These components include the Android 4.0 and IOS 7.0 style components, facilitating your design on mobile apps.

  • Sketch style

    Using the components of the sketch style, every line or element can represent the feel of the rough drawing preliminary sketch, in which you can let free your spirit inspiration as much as you want. The design time is just the coffee time.

  • Review

    Except for the usually used markup components, you can add multiple checks and approvals. Using this function, the team members can note their opinions, which is convenient for feedbacks, collections and arrangements. To use the Free Draw Components, you can also graffiti or punctuate.

  • Export Slide(For Windows Only)

    In addition to export a single page or multiple pages of PNG file and PDF file, you can also export design drawings to the slides. The slides are independent exe files, with no need to install the sof tware and the environment of Mockup Plus, and it can also run independently. When you publish and make presentation of the design drawings, this function will help you.

  • Teamwork (coming soon)

    Registered as a Mockup Plus user, you can use the online team collaboration capabilities. This function will be released soon, please pay close attention to it.
